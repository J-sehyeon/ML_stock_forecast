# ML_stock_forecast
## Overview
이 연구의 핵심 주제는 지식그래프를 사용해 주가를 예측하는 것이다.<br>
팔란티어라는 기업은 온톨로지라는 개념을 사용해 기업, 국가, 국방에 도움을 주는 기업이다.<br>
온톨로지란 객체와 관계, 각각의 속성을 정의한 메타 정보이다. 이를 기반으로 KD: Knowledge Graph 가 생성된다.<br>

팔란티어가 온톨로지를 어떻게 서빙하는지 구체적인 기술이 나와있지 않아 따라할 수는 없다.<br>
하지만 온톨로지라는 기술이 현실 세계의 관계를 디지털로 복사가 가능하다는 사실을 알려줬기 때문에, 이 기술과 llm, 실시간 주가 데이터를 결합한다면 주가의 단기성 변동을 예측할 수 있을 것이라 생각한다.<br>
